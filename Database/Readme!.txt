Cara menambahkan database ke local anda.

1. buka folder 'API' kemudian copy 'KonveksiMaster', pindahkan ke 'htdocs' tempat anda menginstall xampp. contoh directory default xampp : C:\xampp\htdocs

2. buka 'phpmyadmin', jangan lupa mengaktifkan apache dan mysql pada xampp. setelah itu pilih import lalu pilih file 'konveksimaster.sql' dan selesai.
